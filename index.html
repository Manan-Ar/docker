Overview

This project demonstrates containerization of a web application using Docker. Multiple services are isolated into containers and connected using Docker networking.

Technology Stack

  • Node.js Backend
  • PostgreSQL Database
  • Docker
  • Docker Compose

Architecture

Client → Backend Container → Database Container

API

  • POST /data
  • GET /data
  • GET /health

Docker

FROM node:18-alpine
WORKDIR /app
COPY . .
RUN npm install
CMD ["node","server.js"]

Docker Compose

version: "3"
services:
  backend:
    build: .

  db:
    image: postgres

Networking

Macvlan networking allows containers to have their own IP addresses and behave like independent systems.

Key Learning

  • Service isolation
  • Multi-container setup
  • Docker networking
  • Persistent storage
